# Legal RAG Bot

Этот проект представляет собой Telegram-бота для юридического анализа документов с использованием RAG (Retrieval Augmented Generation).

## Функционал

- Индексация юридических документов (PDF, DOCX).
- Поиск релевантных фрагментов по базе знаний (ChromaDB).
- Генерация ответов с юридической аргументацией (Gemini/OpenAI).
- Оптимизирован для работы на системах с ограниченной памятью (8GB RAM).

## Структура проекта

- `bot.py` - Основной код бота.
- `ingest.py` - Скрипт индексации документов (оптимизированный).
- `knowledge_base/` - Папка для исходных документов (игнорируется git).
- `vector_db/` - Векторная база данных (игнорируется git).

## Установка

1. Создайте виртуальное окружение: `python -m venv venv`
2. Активируйте: `venv\Scripts\activate`
3. Установите зависимости: `pip install -r requirements.txt`
4. Создайте файл `.env` с ключами API.

## Запуск

1. Положите документы в `knowledge_base/`.
2. Запустите индексацию: `python ingest.py`
3. Запустите бота: `python bot.py`
